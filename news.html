Дата Заголовок
01.10.2023 Открыта регистрация на Республиканскую олимпиаду по биологии

Регистрация для участия в Республиканской олимпиаде по биологии открыта до 15 октября. Участники могут зарегистрироваться через официальный сайт.
Дополнительная информация будет доступна на нашем сайте.
15.10.2023 Пройдут отборочные этапы для участия в Президентской олимпиаде

Отборочные этапы для Президентской олимпиады начнутся 20 октября. Участники должны подготовить необходимые документы.
Подробности можно узнать у организаторов.
22.10.2023 Объявлены даты проведения Республиканской Юниорской олимпиады

Республиканская Юниорская олимпиада пройдет с 1 по 3 ноября. Участники должны зарегистрироваться до 28 октября.
Следите за новостями для получения актуальной информации.
05.11.2023 Подготовительные семинары для участников олимпиад

Семинары по подготовке участников олимпиад пройдут 10 ноября. Все желающие могут принять участие.
Для регистрации обращайтесь по указанным контактам.
